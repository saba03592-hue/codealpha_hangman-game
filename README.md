# Hangman Game

A simple text-based **Hangman game** written in Python.  
The player guesses a word one letter at a time, with a maximum of 6 incorrect guesses.

## Features

- Small list of predefined words: `apple, banana, orange, grape, mango`
- Limited to 6 incorrect guesses
- Text-based console game (no graphics)
- Tracks guessed letters

## How to Play

1. Run the Python file:
   python hangman.py
2.Guess a letter when prompted.
3.Correct guesses reveal the letter in the word.
4.Wrong guesses are counted. You lose if you reach 6 incorrect guesses.
5.Guess all letters correctly to win!

Requirements
Python 3.x installed
Works on any OS (Windows, Mac, Linux)
